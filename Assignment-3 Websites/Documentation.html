Java

Java is a widely-used object-oriented programming language developed by Sun Microsystems, which is now owned by Oracle. It is used for building cross-platform applications, including desktop, mobile, and web applications.

Java follows the principle of "Write Once, Run Anywhere," meaning that compiled Java code can run on any device that has the Java Runtime Environment (JRE) installed. This makes Java highly portable across different operating systems.

Java is a statically typed language, which means that variable types are defined during compile time, helping catch type-related errors early in the development process.

Java is also known for its rich set of libraries and frameworks that simplify complex tasks, such as Spring for enterprise-level applications and Android SDK for mobile development.

// Simple Java Program public class HelloWorld { public static void main(String[] args) { System.out.println("Hello, World!"); } } // Java Program with Variables public class VariablesExample { public static void main(String[] args) { int number = 5; String message = "The number is: "; System.out.println(message + number); } } // Java Program using If-Else Statement public class IfElseExample { public static void main(String[] args) { int age = 18; if (age >= 18) { System.out.println("You are an adult."); } else { System.out.println("You are a minor."); } } }
JavaScript

JavaScript is a dynamic, high-level programming language used primarily for building interactive websites. It runs in the browser and is widely used for client-side scripting to create dynamic, real-time user experiences.

JavaScript allows for the manipulation of HTML and CSS, enabling dynamic changes to the content and style of a webpage without needing to reload the page. It is essential for modern web development.

Unlike Java, JavaScript is a loosely typed language, meaning that variables do not need to be declared with a specific type. This makes JavaScript flexible but can also lead to runtime errors if not managed carefully.

JavaScript can be used on both the front-end (client-side) and back-end (server-side) of applications, particularly when using Node.js on the server-side to handle requests and responses.

// Simple JavaScript Program let message = "Hello, World!"; console.log(message); // JavaScript Function Example function add(a, b) { return a + b; } console.log(add(5, 10)); // Output: 15 // JavaScript Event Handling Example document.getElementById("myButton").onclick = function() { alert("Button clicked!"); };
